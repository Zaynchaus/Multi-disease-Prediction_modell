
# Multi-Disease Healthcare Prediction System

This project predicts whether a patient is:
- Healthy
- Diabetic
- At risk of Heart Disease

## Tech Stack
- Python
- Streamlit
- Scikit-learn
- Pandas

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Dataset
`multi_disease_dataset.csv`  
Outcome:
- 0 = Healthy
- 1 = Diabetes
- 2 = Heart Disease
